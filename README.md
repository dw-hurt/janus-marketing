# Janus Marketing Website

Marketing website for Janus parental control app beta launch.

## Project Structure

- **content/** - Written content
- **design/** - Mockups and graphics
- **website/** - HTML, CSS, JavaScript
- **backend/** - API and database
- **docs/** - Documentation
- **scripts/** - Automation
